This repository contains original open-source hardware design work, documentation, and supporting materials published by **ALPHA**.

## Original source attribution

If you use, reproduce, modify, manufacture, or distribute work derived from this repository, you must retain attribution to **ALPHA** as the original source in a reasonable and visible manner.

At minimum, attribution should be preserved in:

- Source repositories
- Documentation
- Product pages or sales listings
- Included build or release materials

Recommended attribution language:

**Derived from ALPHA open-source hardware**

or

**Based on original ALPHA design work**

## Modified or rebranded versions

If you sell or distribute modified versions, forks, or reproductions under different branding, those versions should clearly state that they are derived from ALPHA and should not imply that the derivative is an official ALPHA-produced board unless it actually is.

## No removal of origin references

Please do not remove original source references from design files, documentation, or published repository history where they are present.

## Community intent

The purpose of this project is to make obsolete or poorly documented hardware easier to understand and replace in the Honda OBD1 ECU community. Please preserve that spirit when building on this work.
