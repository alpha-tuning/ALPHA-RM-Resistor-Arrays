# Reverse Engineering Notes

The resistor networks documented in this repository were personally reverse engineered by **ALPHA** from original **Honda OBD1 ECU** hardware.

These networks were decoded by working from the original obsolete source components directly. That process involved removing the original parts from OBD1 ECUs, measuring and mapping the internal resistor relationships, identifying the network topology, and then recreating that structure in a way that could be clearly documented and reproduced.

This work was done to help preserve knowledge around hardware that is becoming harder to source, harder to decode, and harder to replace.

## Original source only

ALPHA does **not** reverse engineer other people's reproduced boards, clone boards, or replacement products for the purpose of republishing their work.

These resistor network replacements were derived from the **original obsolete Honda ECU source hardware**, not from another person's recreated version.

That distinction matters.

The goal of this repository is to preserve and document original hardware that would otherwise remain undocumented, unavailable, or locked behind guesswork. This work is intended to support repair, education, fabrication, and open-source hardware development within the OBD1 Honda community.

## Project intent

By publishing this information openly, ALPHA aims to make replacement hardware easier to understand and easier to build.

That includes documenting:

- network topology
- tree and star layouts
- schematics
- assembled PCB views
- Gerbers
- BOMs

The intent is to reduce confusion, reduce gatekeeping, and make obsolete hardware replacement more pain free for the community.

## Respect for original work

ALPHA respects original engineering work and expects the same in return.

This repository documents hardware that was reverse engineered from original OEM-era source components. It is not intended to republish or strip credit from someone else's reproduced design work.
