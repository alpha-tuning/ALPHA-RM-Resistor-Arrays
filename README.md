# ALPHA RM Resistor Networks

Open-source replacement resistor network boards for Honda OBD1 ECUs.

The goal of this repository is to make resistor network replacement as pain-free, understandable, and accessible as possible. Many of the original OBD1 Honda ECU components are now obsolete, difficult to source, undocumented, or simply not decoded well enough for the average builder or repairer to confidently replace. That creates unnecessary friction for people trying to repair ECUs, learn the hardware, or keep these platforms alive.

ALPHA exists to help solve that.

In addition to these resistor network replacements, ALPHA also develops other hardware for OBD1 Honda ECUs, focused on replacing obsolete parts, creating open alternatives, and lowering the barrier for enthusiasts, tuners, and builders who want real hardware they can understand and use. Having replacement hardware for discontinued or poorly documented components is a big deal in this space, and we wanted to approach it in a way that is practical, clear, and fully open source.

This repository is organized to help people do more than just copy a board. It is meant to help people actually understand what they are looking at.

Inside this project you will find resistor network layouts for both **Tree Networks** and **Star Networks**, making it easier to visualize how the internal resistor ladder and branching structures are arranged. Alongside that, the repo includes schematics, assembled PCB views, Gerber files, and BOMs so builders and reverse engineers can move from theory to fabrication without guessing.

Whether you are repairing an ECU, reverse engineering a board, studying resistor array topology, or building your own replacement hardware, this repo is here to give you a clean starting point.

## What this repository includes

This repository contains documentation and build resources for ALPHA resistor network replacement boards, including Tree and Star network styles, with supporting design files and manufacturing outputs.

Included content may contain schematics, PCB layouts, assembled PCB reference views, Gerbers, and BOMs depending on the network and stage of completion.

## Why this matters

Honda OBD1 ECUs rely on a mix of components that are becoming harder to replace every year. Some parts are discontinued. Some are locked inside custom or poorly documented packages. Some are easy to physically remove, but difficult to electrically understand without spending hours decoding them first.

That is where this project comes in.

By documenting these resistor networks clearly and releasing the work openly, ALPHA is trying to remove the mystery and make the replacement process easier for everyone. Less gatekeeping, less guesswork, and more working hardware.

## Open-source intent

This project is part of the broader ALPHA mission to create practical, affordable, open hardware for the OBD1 Honda community.

The intent is simple: make replacement hardware available, make it understandable, and make it easier for others to build on the work.

## Notes

These files are provided to support repair, learning, fabrication, and further development. Always verify fit, function, and electrical compatibility for your specific ECU application before installation.
